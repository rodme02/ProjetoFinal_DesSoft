# NIGHT RUNNER

Jogo de pular obstáculos, cujo objetivo é conseguir um score alto. A velocidade do jogo aumenta com o tempo para o jogo ir dificultando.

Para executar o jogo rode o arquivo pygame_projetofinal.py
